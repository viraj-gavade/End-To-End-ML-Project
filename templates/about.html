About This Project

Project Overview

The Student Performance Prediction application is a machine learning-based tool designed to predict students' math scores based on various factors. This project aims to help educators, parents, and students understand the impact of different variables on academic performance.

Purpose

By analyzing patterns in student data, we can identify key factors that influence academic success and provide targeted interventions to improve learning outcomes.

The Dataset

This project is built using the "Student Performance Dataset" which contains information about:

  • Gender (male/female)
  • Race/ethnicity (groups A-E)
  • Parental level of education
  • Lunch type (standard or free/reduced)
  • Test preparation course completion status
  • Reading, writing, and math scores

Machine Learning Model

We've trained several regression models on the dataset and selected the best performing one based on accuracy metrics. Our model analyzes the relationship between various factors and math scores to make accurate predictions.

Technologies Used

  • Python: Core programming language
  • Scikit-learn & XGBoost: For building machine learning models
  • Pandas & NumPy: For data manipulation
  • Matplotlib & Seaborn: For data visualization
  • Flask: Web framework for the application

Project Development

This project was developed following these steps:

  1. Exploratory Data Analysis (EDA) to understand patterns and relationships
  2. Data preprocessing and feature engineering
  3. Model selection and hyperparameter tuning
  4. Web application development using Flask
  5. Deployment and monitoring

Future Improvements
